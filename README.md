MAX44099 I2C  Driver
====

-  Read data finish  with Softwere I2C.
- Add connect to  high(3.3V)

## Connect

SCL-> D12

SDA-> D11

ADDR->3.3V

VCC->3.3V or 5.0V

GND->GND
